# To-Do List App

This is a simple web-based To-Do List app built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Stores tasks in browser using localStorage

## How to Run
Open `index.html` in any browser.
